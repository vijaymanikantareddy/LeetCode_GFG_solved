<h2><a href="https://www.geeksforgeeks.org/problems/range-sum-of-bst/1">Sum of Nodes in BST Range</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given the <strong>root </strong>of a Binary Search Tree and two integers <strong>l </strong>and <strong>r</strong>, the task is to find the <strong data-start="367" data-end="439">sum of all nodes</strong> that lie between l and r, including both l and r.</span></p>
<p><span style="font-size: 14pt;"><strong>Examples</strong></span></p>
