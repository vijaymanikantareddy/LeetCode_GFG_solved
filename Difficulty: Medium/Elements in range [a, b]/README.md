<h2><a href="https://www.geeksforgeeks.org/problems/find-number-of-elements-in-range-a-b-for-each-query/1">Elements in range [a, b]</a></h2><h3>Difficulty Level : Difficulty: Medium</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 14pt;">Given an unsorted array <strong>arr[]</strong> and a 2D array <strong>queries[][] </strong>of size <strong>q</strong>, where each query is in the form of [a, b]. For each query, <strong>count</strong> how many elements in arr[] lie within the range <strong>[a, b]</strong>, i.e., elements satisfying <strong>a ≤ x ≤ b</strong>.</span></p>
<p><span style="font-size: 14pt;">Return the results for all queries as a list of integers, where each integer corresponds to the count of elements in the respective query range.</span></p>
<p><strong><span style="font-size: 14pt;">Examples:</span></strong></p>
<pre><span style="font-size: 14pt;"><span style="font-size: 14pt;"><strong>Input: </strong></span><span style="font-size: 18.6667px;">arr[] = [1, 4, 2, 8, 5], queries[][] = [[1, 4], [3, 6], [0, 10]]</span><strong style="font-size: 14pt;"><br>Output: </strong><span style="font-size: 18.6667px;">[3, 2, 5]</span><strong style="font-size: 14pt;"><br>Explanation: </strong><span style="font-size: 18.6667px;">Query [1, 4]: Elements in range → [1, 4, 2] → Count = 3
Query [3, 6]: Elements in range → [4, 5] → Count = 2
Query [0, 10]: All elements → [1, 4, 2, 8, 5] → Count = 5</span></span></pre>
<pre><span style="font-size: 14pt;"><span style="font-size: 14pt;"><strong>Input: </strong></span><span style="font-size: 18.6667px;">arr[] = [10, 20, 30, 40, 50], queries[][] = [[5, 15], [25, 45], [10, 50]]</span><strong style="font-size: 14pt;"><br>Output: </strong><span style="font-size: 18.6667px;">[1, 2, 5]</span><strong style="font-size: 14pt;"><br>Explanation:</strong><span style="font-size: 14pt;"> </span></span><span style="font-size: 18.6667px;">Query [5, 15]: Elements in range → [10] → Count = 1
Query [25, 45]: Elements in range → [30, 40] → Count = 2
Query [10, 50]: Elements in range → [10, 20, 30, 40, 50] → Count = 5<br></span></pre>
<p><span style="font-size: 18.6667px;"><strong>Constraints:</strong><br>1 ≤ arr.size(), q ≤ 10<sup>5</sup><br>0 ≤ arr[i] ≤ 10<sup>6</sup><br>0 ≤ queries[i][0] ≤ queries[i][1] ≤ 10<sup>6</sup></span></p></div><br><p><span style=font-size:18px><strong>Topic Tags : </strong><br><code>Binary Search</code>&nbsp;<code>Algorithms</code>&nbsp;
